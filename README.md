# 🤖 基于Hamibot 的闲鱼微信小程序自动确认收款脚本

> 一款基于 **[Hamibot](https://hamibot.com/)** 的安卓自动化脚本，用于自动识别并点击应用中的「确认收款」按钮，自动完成从收款到完成的全流程操作。  
> 适用于需要频繁手动确认收款的场景，让操作更加智能、高效。

---

## 🚀 功能简介

- 自动识别屏幕上的 **「确认收款」** 按钮  
- 自动执行完整的 **收款 → 完成** 流程  
- 支持 **循环执行**，直到无待处理收款项  
- 自动跳过顶部“未结算”卡片，避免误操作  
- 控制台日志实时输出，便于调试和监控  

---

## 📱 运行环境

| 项目 | 说明 |
|------|------|
| **系统要求** | Android 7.0 及以上 |
| **依赖工具** | [Hamibot App](https://hamibot.com/download) |
| **脚本语言** | JavaScript（Auto.js 兼容语法） |

---

## ⚙️ 使用方法

1. **安装 Hamibot 应用**  
   前往 [Hamibot 官网](https://hamibot.com/) 下载并安装最新版本。

2. **创建新脚本**  
   登录 Hamibot 控制台 → 新建脚本 → 将主脚本代码粘贴进去。

3. **授予权限**  
   - 启用辅助功能权限  
   - 启用无障碍服务  
   - 允许显示悬浮窗（用于日志窗口）

4. **运行脚本**  
   在目标应用界面启动脚本，Hamibot 将自动执行收款流程。

## 🧠 调试与优化

- 建议保持日志窗口开启 (`console.show()`)，方便查看运行状态。
